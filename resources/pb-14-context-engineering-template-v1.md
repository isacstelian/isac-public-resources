# CONTEXT ENGINEERING - TEMPLATE COPY-PASTE

Resursă originală ISAC / AI Society România · v1 · keyword `CONTEXT`

Folosește acest template când vrei un rezultat important de la Claude, ChatGPT sau alt agent AI. Completează doar informațiile pe care le cunoști. Dacă ceva lipsește, cere agentului să întrebe înainte să lucreze.

```text
REZULTAT
Vreau să obțin: [livrabilul concret].
Îl voi folosi pentru: [scop / public / canal].

CONTEXT
Situația actuală: [ce există deja].
Ce am încercat: [opțional].
Ce contează cel mai mult: [priorități].

REGULI
Trebuie să: [reguli obligatorii].
Nu trebuie să: [interdicții].
Ton / stil: [exemple simple].

MATERIALE
Folosește aceste fișiere / linkuri / exemple: [listă].
Sursa de adevăr este: [fișier / document / URL].
Nu inventa informații care nu apar în surse.

MOD DE LUCRU
1. Citește materialele.
2. Spune ce ai înțeles și ce lipsește.
3. Pune întrebări dacă o lipsă schimbă rezultatul.
4. Execută în pași simpli.
5. Arată ce ai verificat.

VERIFICARE
Consider rezultatul gata doar dacă:
- [criteriul 1]
- [criteriul 2]
- [criteriul 3]

La final livrează: [formatul exact].
Dacă nu poți verifica ceva, marchează clar „NEVERIFICAT”.
```

## Sistemul în 6 straturi

1. **Rezultat** - descrie ce trebuie să existe la final.
2. **Context** - explică situația, publicul și motivul.
3. **Reguli** - spune ce este obligatoriu și ce este interzis.
4. **Exemple** - arată unul bun și, dacă ajută, unul greșit.
5. **Fișiere + skills** - oferă sursa de adevăr și instrucțiunile reutilizabile. Un skill este o procedură salvată pentru un task repetat.
6. **Verificare** - definește testul care dovedește că munca este gata.

## Regula de siguranță

Nu încărca parole, chei API, CNP, date bancare sau documente confidențiale într-un serviciu AI fără aprobarea și protecțiile potrivite. Pentru informații importante, cere surse și verifică rezultatul.
